# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题
###### 六级标题

Markdown 段落

段落编写完成后，
在末尾增加两个以上的 space空格 然后 enter。

段落编写完成后，

也可在编写的末尾增加一行空行，进行划分段落。

Markdown 字体

斜体文本 *斜体文本*

斜体文本 _斜体文本_

粗体文本 **粗体文本**

粗体文本 __粗体文本__

粗斜体文本 ***粗斜体文本***

粗斜体文本 ___粗斜体文本___

Markdown 分隔线

三个及以上的*，-，或者三个以上的_。

* * *

- - -

_ _ _

但是行内不能添加东西，可以在*，-，或者_中间添加space空格。

Markdown 删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上~~ 两个波浪线 即可

利用HTML的 <u> 标签

带下划线文本

Markdown 脚注

Markdown 脚注的格式如下:

[^要注明的文本]

[^要注明的文本]

Markdown 列表

Markdown 支持有序列表和无序列表。

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记：

* 第一项
* 第二项
* 第三项

  • 第一项
  • 第二项
  • 第三项

+ 第一项
+ 第二项
+ 第三项

  • 第一项
  • 第二项
  • 第三项

- 第一项
- 第二项
- 第三项

  • 第一项
  • 第二项
  • 第三项

有序列表使用数字并加上 . 号来表示:

1. 第一项
2. 第二项
3. 第三项

  1. 第一项
  2. 第二项
  3. 第三项

Markdown 列表嵌套

列表嵌套

列表嵌套只需在子列表中的选项添加四个空格即可：

  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素
  3. 第三项
    • 第三项嵌套的第一个元素
    • 第三项嵌套的第二个元素

Markdown 区块

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个 space空格 符号：

区块引用
示例教程
区块嵌套

另外区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：

最外层

第一层嵌套

第二层嵌套

Markdown 列表中使用区块

如果要在列表项目内放进区块，那么就需要在 > 前添加四个space空格的缩进。

区块中使用列表实例如下：

  • 第一项

    菜鸟教程
    学的不仅是技术更是梦想

  • 第二项
